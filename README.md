# Интерпретатор машины Тьюринга

## Синтаксис

Правила, которые будут подаваться на вход интерпретатору, должны выглядеть следующим образом:
```
<state>,<symbol> -> <new_state>,<new_symbol>,<move>
```

`state` и `new_state` - старое и новое состояния (строки из любых нормальных ascii символов кроме пробельных и `,`). 

`symbol` и `new_symbol` - старый и новый символы, записанные на ленте (требования те же, что и к состояниям)

`move` - `L`, `R`, или `N` (то, куда сдвинется указатель).

Изначально машина находится в состоянии `_start`, когда машина пришла в состояние `_finish` вычисление считается завершенным. 
Изначально указатель расположен в начале входа. 
Во всех остальных ячейках ленты записано `#`.
Результатом исполнения считается строка от позиции указателя в состоянии `_finish` до первой `#`.

Можно оставлять пустые строки для выделения логических блоков кода.

Примеры кода есть в `examples`.

## Как запустить

- Склонировать репозиторий

```
python main.py <filename>
```

`filename` - имя файла с кодом.

Далее программа будет ожидать вход: строку символов, все символы разделены пробелами!!! (для поддержки символов из нескольких букв).

### Пример
```
python main.py examples/inverter

stdin:
0 1 0 1

stdout:
1 0 1 0
```
