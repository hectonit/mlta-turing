# Интерпретатор машины Тьюринга

## Синтаксис

Правила, которые будут подаваться на вход интерпретатору, должны выглядеть следующим образом:
```
<state>,<symbol> -> <new_state>,<new_symbol>,<move>
```

`state` и `new_state` - старое и новое состояния (строки из любых нормальных ascii символов кроме пробельных и `,`). 

`symbol` и `new_symbol` - старый и новый символы, записанные на ленте (требования те же, что и к состояниям)

`move` - `L`, `R`, или `N` (то, куда сдвинется указатель).

Изначально машина находится в состоянии `_start`, когда машина пришла в состояние `_finish` вычисление считается завершенным. Изначально на ленте везде написаны `#`.

Вход указывается первой строкой файла. Примеры кода есть в `examples`.
